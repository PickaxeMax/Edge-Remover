# Скрипт для удаления Edge 🚫🌐

Простой, быстрый и эффективный `.bat` скрипт для полного удаления браузера Microsoft Edge из системы, очистки оставшихся ярлыков и блокировки его повторной установки через Центр обновлений Windows.

## ✨ Возможности скрипта
1. **Принудительное закрытие:** Завершает все активные фоновые процессы Microsoft Edge.
2. **Глубокая очистка:** Безвозвратно удаляет папки программы и основные системные файлы.
3. **Очистка кэша:** Стирает пользовательские данные, профили и кэш браузера.
4. **Удаление ярлыков:** Вычищает мертвые ярлыки с Рабочего стола, из меню «Пуск» и Панели задач.
5. **Блокировка обновлений:** Добавляет официальный ключ в реестр, который запрещает Центру обновлений Windows заново скачивать Edge.

## 🚀 Инструкция по использованию

1. Скачайте файл `edgeremover.bat` из этого репозитория.
2. Нажмите по скачанному файлу правой кнопкой мыши.
3. Выберите **Запуск от имени администратора** (это необходимо для удаления системных файлов и работы с реестром).
4. Дождитесь сообщения об успешном завершении и нажмите любую клавишу, чтобы закрыть окно.

## EN:
# Edge Remover Script 🚫🌐

A simple, fast, and effective Windows `.bat` script to completely uninstall the Microsoft Edge browser, wipe its leftover shortcuts, and block it from re-installing via Windows Update.

## ✨ Features
1. **Force Close:** Terminates all active background processes of Microsoft Edge.
2. **Deep Clean:** Forcefully deletes application folders and core system files.
3. **Cache Wipe:** Clears user-specific data, profiles, and browser cache.
4. **Shortcut Purge:** Removes leftover shortcuts from the Desktop, Start Menu, and Taskbar.
5. **Update Block:** Adds a registry key that officially tells Windows Update to skip Microsoft Edge installation.

## 🚀 How to Use

1. Download the `remove_edge.bat` file from this repository.
2. Right-click on the downloaded file.
3. Select **Run as administrator** (required to modify system files and registry).
4. Wait for the success message and press any key to close the window.

## 🛠️ Requirements
* Windows 10 or Windows 11
* Administrator privileges
